# Hermes 运维技能 ⚙️

Profile 铁律 + CLI 卡死恢复 + TUI 调试。

## 内容
- 建 Profile 铁律：先计划/定点修复/改前备份
- CLI 卡死恢复：OSError 根因 + tmux 方案
- TUI 命令调试：Python → Gateway → Ink UI 三层诊断

## 安装
```bash
git clone https://github.com/remix2111/hermagent-ops.git
cp -r hermagent-ops/skills/hermes-ops ~/.hermes/skills/
```

## 使用
